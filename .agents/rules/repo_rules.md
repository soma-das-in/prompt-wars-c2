# Repository Rules

The repository must remain under 1 MB.

Rules:

- Avoid large dependencies.
- Do not include large datasets.
- Do not include large assets or images.
- Avoid frontend frameworks such as React or Angular.
- Use lightweight HTML and CSS.

Allowed:

- FastAPI
- minimal Python libraries
- simple static files

Project structure must remain small and clear.
